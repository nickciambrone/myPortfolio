Jeopardy Game

This application takes the user through a simulation of real Jeopardy. The user enters his/her name, and is then given a board of dollar values. The user selects a dollar value under a specific category and is prompted with a question. They must answer the question correctly before time runs out in order to earn the points. If they do not answer the question correctly the dollar value that they chose is deducted from their score. The API calls that my group members and I used are rendered by the browser as "unsafe scripts", and so the user must allow the broswer to load the unsafe scripts in order for the application to function. This can be done in chrome settings.
